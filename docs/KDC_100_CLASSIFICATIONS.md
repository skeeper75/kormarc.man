# KDC 100개 분류 선정 보고서

**작성일**: 2026-01-10
**KDC 버전**: 6판 (2013)
**총 분류 수**: 100개 (각 주류별 10개)

---

## 1. 개요

본 문서는 한국십진분류표(KDC) 3자리 분류 중 데이터 수집을 위한 100개 분류를 선정한 결과를 담고 있습니다.

### 1.1 선정 기준

1. **균등 배분**: 각 주류(0-9)에서 10개씩 선정하여 총 100개 확보
2. **인기도**: 도서관 장서가 많고 이용률이 높은 분야 우선
3. **한국 맥락**: 한국 독자와 도서관에 중요한 주제 우선
4. **수집 가능성**: API/스크래퍼로 데이터 수집이 가능한 주제
5. **다양성**: 각 주류 내에서 다양한 세부 주제 포함

### 1.2 난이도 분류

- **Easy (40개)**: 일반 공공도서관에서 쉽게 찾을 수 있는 주제
- **Medium (48개)**: 전문 도서관이나 대학 도서관이 필요한 주제
- **Hard (12개)**: 특수 컬렉션이나 전문 기관이 필요한 주제

## 4. 데이터 수집 전략 권장사항

**목표**: 초기 시스템 검증 및 기본 데이터셋 구축
**기간**: 1-2개월
**방법**: 공공도서관 API, 일반 서점 스크래핑

**우선 순위 분류**:
1. 한국문학 (810, 811, 813, 814): 가장 많은 장서

4. 컴퓨터/IT (004, 005): 최신 자료 많음
5. 역사/전기 (911, 912, 980): 인기 분야

### 4.2 Phase 2: Medium 분류 확장 (48개)

**목표**: 데이터셋 다양성 확보
**기간**: 2-3개월
**방법**: 대학 도서관 협력, 전문 서점 활용

**우선 순위 분류**:
1. 의학 세부 분류 (511, 513, 517): 전문 자료
2. 기술과학 (530, 550): 공학 도서관

4. 종교 세부 분류 (231, 232, 240): 종교 기관 협력

## 6. 품질 검증 체크리스트

### 6.1 수집 데이터 검증 항목

각 분류별 데이터 수집 시 다음 항목을 확인합니다:

- [ ] KDC 코드가 정확하게 매핑되었는가?
- [ ] ISBN이 유효한가?
- [ ] 서지 정보가 완전한가? (제목, 저자, 출판사, 출판년도)
- [ ] 중복 레코드가 없는가?
- [ ] 검색 키워드와 분류가 일치하는가?

### 6.2 분류별 최소 수집 목표

- **Easy 분류**: 분류당 최소 100건
- **Medium 분류**: 분류당 최소 50건
- **Hard 분류**: 분류당 최소 20건
