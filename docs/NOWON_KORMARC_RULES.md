# 노원구립도서관 KORMARC 규칙

## 개요

본 문서는 노원구립도서관의 KORMARC (한국문헌자동화목록형식) 적용 규칙을 정리한 것입니다.

## 기준 문서

- **KORMARC 표준**: KS X 6006-0 (2014개정, 2023개정)
- **기관 코드**: NLK (국립중앙도서관)
- **시방서**: 노원구립도서관 연간단가계약제안서 (2025년)

## 노원구 특화 규칙

### 목록작성기관 (040 필드)

```
040   ##$aNLK$bkor$c(NLK)$dNLK$eKORMARC2014
```

- `040$a`: 국립중앙도서관 (NLK)
- `040$b`: 목록 언어 (kor)
- `040$c`: 전사기관 ((NLK))
- `040$d`: 수정기관 (NLK)
- `040$e`: 기술규칙 (KORMARC2014)

### ISBN 입력 (020 필드)

```
020   ##$a9791162233149
```

- ISBN 13자리를 기본으로 사용
- 하이픈(-) 제거 후 입력

### 도서형태 (008 필드)

```
008   150508s2025    ko  a
```

- 위치 06-07: `a` (문학)
- 위치 21: `m` (단행본)
- 위치 33-35: `kor` (한국)

## 노원구 분류 기호

### KDC (한국십진분류표)

- **0**: 총류
- **1**: 철학
- **2**: 종교
- **3**: 사회과학
- **4**: 자연과학
- **5**: 기술과학
- **6**: 예술
- **7**: 언어
- **8**: 문학
- **9**: 역사

## 레코드 생성 순서

1. **ISBN 수집**: 도서 ISBN 확보
2. **도서 정보 조회**: 제목, 저자, 발행자, 발행년 등
3. **KORMARC 필드 매핑**: 표준 필드에 데이터 할당
4. **TOON 식별자 생성**: `kormarc_book_[ULID]`
5. **JSON 변환**: 저장용 JSON 포맷 변환
